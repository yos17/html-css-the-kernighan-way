Chapter 5 — Grid System

Interactive playground for CSS Grid concepts: 12-column layout, named areas, auto-fill vs auto-fit, and more.

1. The 12-Column Grid

Adjust the number of columns and gap. Click cells to toggle spanning.

12 8px 
.grid {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 8px;
}

2. Named Template Areas

The page layout below uses grid-template-areas. Toggle mobile mode to see the responsive rearrangement.

HEADER
SIDEBAR
CONTENT
FOOTER
.page-layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: 50px 1fr 40px;
  grid-template-areas:
    "header  header"
    "sidebar content"
    "footer  footer";
}

3. auto-fill vs. auto-fit

The key difference shows when there are fewer items than columns. Resize to see it.

100px 3
auto-fill — empty tracks remain
auto-fit — empty tracks collapse
/* auto-fill — items stay at min size, empties remain */
.grid { grid-template-columns: repeat(auto-fill, minmax(100px, 1fr)); }

/* auto-fit — items expand to fill, empties collapse */
.grid { grid-template-columns: repeat(auto-fit, minmax(100px, 1fr)); }

4. Dense Auto-Placement

grid-auto-flow: dense fills gaps left by spanning items. Toggle it to see the difference. 

.grid {
  grid-auto-flow: dense; /* fill gaps with later items */
  grid-auto-rows: 60px;
}
