2. Transitions vs @keyframes

Transition — Hover to Animate

State-driven. Reverses when you stop hovering.

🔲
🔄
🎨
⭐
.box { transition: transform 300ms ease-out; } .box:hover { transform: scale(1.15); }

@keyframes — Click to Trigger

Standalone motion. Does not reverse automatically.

🎯
🌀
❤
🪀
.box { animation: bounceIn 500ms both; } /* applied once; use .animated class */

10. Rendering Pipeline Reference

  ┌─────────────────────────────────────────────────────────────────────────────┐
  │                         RENDERING PIPELINE                                  │
  ├─────────────┬─────────────┬──────────────┬─────────────────────────────────┤
  │   STYLE     │   LAYOUT    │    PAINT     │         COMPOSITE               │
  │             │             │              │                                  │
  │  Resolve    │  Calc box   │  Fill in     │  GPU assembles layers           │
  │  all CSS    │  positions  │  pixels:     │  into final frame               │
  │  values     │  and sizes  │  color, bg,  │                                 │
  │             │             │  shadows     │  Only transform + opacity       │
  │             │             │              │  land here directly             │
  ├─────────────┼─────────────┼──────────────┼─────────────────────────────────┤
  │  cheap      │  EXPENSIVE  │   MEDIUM     │  free (GPU only)                │
  └─────────────┴─────────────┴──────────────┴─────────────────────────────────┘

  Animating:   left/top/width/height  → triggers Layout  ❌ avoid
               color/background       → triggers Paint   ⚠ acceptable
               transform/opacity      → Composite only   ✓ always prefer
