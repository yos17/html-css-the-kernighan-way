em vs rem — The Nesting Problem
em is relative to the current element's font-size — it compounds in nested elements. rem is always relative to the root element — it never compounds. Use rem for font sizes, em for padding and borders that should scale with local text size.

Vertical Rhythm — Consistent Spacing
Chapter 2: Typography

Typography is not decoration. It is the primary tool through which a user interface communicates. Understanding the underlying system — font stacks, type scales, and vertical rhythm — is what separates professional CSS from guesswork.

The Type Scale

A modular type scale applies a constant ratio between each step. Every size in the system is the base multiplied by the ratio raised to some power. Change the base or the ratio and the entire system rescales consistently.

The 1.25 ratio (Major Third) creates clear hierarchy without extreme size differences. At display sizes the hierarchy is obvious; at body size the relationship reads as organized without being dramatic.

Vertical Rhythm

Consistent spacing is computed as multiples of the base rhythm unit (font-size × line-height). Headings get 2× before them and 0.5× after. Paragraphs are separated by 0.5× rhythm. The result is a page where the eye moves naturally from section to section.

type-system.css — Complete Listing
/* type-system.css — HTML & CSS: The Kernighan Way */ html { font-size: 62.5%; } /* 1rem = 10px at default browser settings */ body { font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif; font-size: 1.6rem; /* 16px */ line-height: 1.6; } code, pre, kbd, samp { font-family: 'JetBrains Mono', 'Fira Code', 'Courier New', monospace; } code { font-size: 0.9em; } /* em scales with surrounding text */ .prose { max-width: 68ch; } /* optimal reading line length */ /* Fluid type: clamp(min, viewport-expression, max) */ h1 { font-size: clamp(3.2rem, 5vw + 1rem, 6.104rem); font-weight: 700; line-height: 1.15; letter-spacing: -0.02em; } h2 { font-size: clamp(2.8rem, 4vw + 0.8rem, 4.883rem); font-weight: 700; line-height: 1.2; letter-spacing: -0.01em; } h3 { font-size: clamp(2.2rem, 3vw + 0.5rem, 3.906rem); font-weight: 600; line-height: 1.25; } h4 { font-size: clamp(1.8rem, 2vw + 0.3rem, 3.125rem); font-weight: 600; line-height: 1.3; } h5 { font-size: clamp(1.6rem, 1.5vw + 0.2rem, 2.5rem); font-weight: 600; line-height: 1.35; } h6 { font-size: clamp(1.4rem, 1vw + 0.2rem, 2.0rem); font-weight: 600; line-height: 1.4; } small { font-size: 1.28rem; line-height: 1.5; } :root { --rhythm: 2.56rem; } /* 1.6rem × 1.6 line-height */ h1, h2, h3, h4, h5, h6 { margin-top: calc(var(--rhythm) * 2); margin-bottom: calc(var(--rhythm) * 0.5); } p + p { margin-top: calc(var(--rhythm) * 0.5); }
