Auto-Fit Card Grid

repeat(auto-fit, minmax(180px, 1fr)) — no media queries needed. Columns fill and collapse automatically.

Named Template Areas

The "holy grail" layout — header, two sidebars, main content, footer — in 10 lines of CSS using grid-template-areas.

header
sidebar
grid-area: side
main content
grid-area: main
aside
grid-area: asid
footer
.holy-grail {
  display: grid;
  grid-template-columns: 150px 1fr 120px;
  grid-template-areas:
    "hdr  hdr  hdr "
    "side main asid"
    "ftr  ftr  ftr ";
}
