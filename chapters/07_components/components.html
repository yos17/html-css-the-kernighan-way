4. Alerts

Four semantic alert types. Click the × to dismiss with a height transition.

ℹ
Information
This is an informational alert. It uses currentColor so the icon and close button inherit the text color automatically.
✓
Success
Your changes have been saved. The operation completed without errors.
⚠
Warning
Your session will expire in 5 minutes. Save your work to avoid losing changes.
✕
Error
Failed to connect to the server. Check your network connection and try again. 
.alert--info  { background: #0c1a2e; color: #93c5fd; }
/* icon and close button get the same color automatically: */
.alert__icon  { color: currentColor; }
.alert__close { color: currentColor; }
