7. Stacking Context Reference

STACKING CONTEXT CREATORS (complete list)
──────────────────────────────────────────────────────────────────
  Property                   When it creates a stacking context
  ─────────────────────────  ─────────────────────────────────────
  opacity                    any value < 1
  transform                  any value other than none
  filter                     any value other than none
  backdrop-filter            any value
  perspective                any value other than none
  clip-path                  any value other than none
  mask / mask-image          any value
  isolation                  isolate
  mix-blend-mode             any value other than normal
  will-change                transform | opacity | filter | etc.
  contain                    layout | paint | strict | content
  z-index                    any value other than auto,
                             AND position ≠ static
  position                   fixed or sticky (always)
──────────────────────────────────────────────────────────────────

COMMON ACCIDENTAL CONTEXTS:
  transform: translateZ(0)   GPU promotion trick → stacking context
  will-change: transform     Animation hint → stacking context
  opacity: 0.99              Fade trick → stacking context
  filter: blur(0)            "Sharpen" trick → stacking context
