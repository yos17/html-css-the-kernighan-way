The Box Model

Every HTML element is a rectangle with four layers. The critical choice: does width include padding and border (border-box) or just the content (content-box, the browser default)?

Inheritance

Text properties inherit down the DOM tree. Box properties don't. Form elements block font inheritance — that's why input, button { font: inherit } is in every reset.

The Complete Reset

Six rules. Each earns its place.

@layer reset {
  /* 1. Box model: width includes padding + border */
  *, *::before, *::after {
    box-sizing: border-box;
  }

  /* 2. Remove browser default margins */
  * { margin: 0; }

  /* 3. Body baseline */
  body {
    min-height: 100dvh;
    line-height: 1.5;
    -webkit-font-smoothing: antialiased;
  }

  /* 4. Media: block removes the 4px descender gap */
  img, picture, video, canvas, svg {
    display: block;
    max-width: 100%;
  }

  /* 5. Forms: force font inheritance (browsers block it) */
  input, button, textarea, select {
    font: inherit;
  }

  /* 6. Prevent text overflow on long words */
  p, h1, h2, h3, h4, h5, h6 {
    overflow-wrap: break-word;
  }
}
