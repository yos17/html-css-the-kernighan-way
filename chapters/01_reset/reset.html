Box Model — content-box vs border-box
Both boxes below are declared with width: 200px; padding: 20px; border: 8px solid. Watch what happens to the rendered width.

/* content-box: padding and border ADD to the width */ .box { box-sizing: content-box; /* browser default */ width: 200px; /* only the content box */ padding: 20px; /* 20px × 2 sides = 40px extra */ border: 8px solid; /* 8px × 2 sides = 16px extra */ } /* Rendered width: 200 + 20 + 20 + 8 + 8 = 256px */ /* Two of these side-by-side in a 500px container: */ /* 256 + 256 = 512px — overflows by 12px! */
/* border-box: padding and border are INCLUDED in the width */ .box { box-sizing: border-box; width: 200px; /* content + padding + border = 200px */ padding: 20px; /* uses 20px × 2 = 40px from the 200px */ border: 8px solid; /* uses 8px × 2 = 16px from the 200px */ } /* Content area: 200 - 40 - 16 = 144px */ /* Rendered width: 200px (always) */ /* Two 50%-wide boxes always fit side by side */
/* Classic layout bug with content-box: */ .sidebar { width: 25%; padding: 20px; } /* renders at 25% + 40px */ .main { width: 75%; padding: 20px; } /* renders at 75% + 40px */ /* Total: 100% + 80px — wraps to next line! */ /* The fix (v1 of micro-reset.css): */ *, *::before, *::after { box-sizing: border-box; } /* Now: sidebar is exactly 25%, main is exactly 75% */ /* They fit. Every time. */
Specificity — How Rules Win
Every CSS selector has a specificity score: (IDs, Classes, Types). Higher wins. When equal, the later rule wins. The universal selector * scores (0,0,0) — the reset always loses to your rules.

Inheritance — What Flows Down the DOM
Typography properties inherit. Layout properties do not. Form elements block font inheritance — that's why font: inherit is essential in every reset.
body font-family: 'Courier New' declared
└─ p font-family: 'Courier New' inherited ✓
└─ a font-family: 'Courier New' inherited ✓
└─ button font-family: system-ui UA override ✗
└─ button + reset font-family: 'Courier New' font: inherit ✓
└─ input font-family: system-ui UA override ✗
└─ input + reset font-family: 'Courier New' font: inherit ✓
body background-color: #0d0d1a declared
└─ p background-color: transparent NOT inherited ✗
micro-reset.css — Complete Listing
/* v1: The one rule that matters */ *, *::before, *::after { box-sizing: border-box; /* padding+border included in width */ margin: 0; /* no browser margin surprises */ padding: 0; /* no browser padding surprises */ } /* 3 rules. Fixes the most common layout bugs. */ /* Cost: all elements now have no spacing — add it back deliberately. */
/* v2: Element-specific resets */ *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; } /* Headings — remove browser font-size/weight decisions */ h1, h2, h3, h4, h5, h6 { font-size: inherit; font-weight: inherit; } /* Lists — remove bullets and indent */ ul, ol { list-style: none; } /* Links — remove blue color and underline */ a { color: inherit; text-decoration: inherit; } /* Media — block display removes 4px descender gap */ img, video, canvas, svg { display: block; max-width: 100%; } /* Forms — font: inherit is critical. Browsers block font inheritance here. */ button, input, select, textarea { font: inherit; /* forces UA-blocked inheritance */ color: inherit; background: none; border: none; }
/* v3: micro-reset.css — production-ready, ~40 lines */ *, *::before, *::after { box-sizing: border-box; margin: 0; padding: 0; } html { -webkit-text-size-adjust: 100%; /* prevent iOS font inflation on rotate */ tab-size: 4; scroll-behavior: smooth; } body { min-height: 100vh; line-height: 1.5; -webkit-font-smoothing: antialiased; text-rendering: optimizeLegibility; } h1, h2, h3, h4, h5, h6 { font-size: inherit; font-weight: inherit; } p, h1, h2, h3, h4, h5, h6 { overflow-wrap: break-word; } ul, ol { list-style: none; } a { color: inherit; text-decoration: inherit; } img, picture, video, canvas, svg { display: block; max-width: 100%; } img, video { height: auto; } button, input, select, textarea, optgroup { font: inherit; color: inherit; background: none; border: none; padding: 0; } button, [type="button"], [type="reset"], [type="submit"] { cursor: pointer; -webkit-appearance: button; } button:disabled { cursor: default; } textarea { resize: vertical; } input[type="search"] { -webkit-appearance: none; outline-offset: -2px; } table { border-collapse: collapse; border-spacing: 0; } summary { cursor: pointer; } [hidden] { display: none !important; } :focus-visible { outline: 2px solid currentColor; outline-offset: 2px; }
