5. Cascade Layers (@layer)

Layers determine which styles win — regardless of specificity. Toggle classes to see layers compete.

This element's style is determined by layers.
Layer priority (lowest → highest):
reset → base → components → utilities → unlayered (highest!) 
@layer reset, base, components, utilities;

@layer base {
  .element { color: #666; }
}

@layer components {
  .element.card { color: #eee; background: #16213e; }
  /* specificity: 0-2-0 — normally beats utilities */
}

@layer utilities {
  .text-accent { color: #a8d8ea; }
  /* specificity: 0-1-0 — but utilities layer wins! */
}

6. Viewport Units: vh vs dvh

On mobile browsers, the address bar hides on scroll — changing the visible height. dvh tracks the dynamic visible height.

vw — viewport width
1vw = —
50vw = —
vh — viewport height
1vh = —
100vh = —
dvh — dynamic height
Tracks visible area
Use for mobile full-height
svh/lvh
svh = small (UI visible)
lvh = large (UI hidden) 
/* Broken on mobile — vh doesn't account for browser UI */
.hero { height: 100vh; }

/* Correct — dvh tracks actual visible height */
.hero {
  height: 100vh;   /* fallback */
  height: 100dvh; /* dynamic viewport height */
}
